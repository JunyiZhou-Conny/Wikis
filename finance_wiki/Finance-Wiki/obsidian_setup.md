# Obsidian setup — finance_wiki

1. Install [Obsidian](https://obsidian.md) v1.12.4+
2. *Open folder as vault* → `finance_wiki/Finance-Wiki/` (**not** repo root)
3. Start at [[Home MOC]]
4. Enable **Dataview** community plugin

## Sources layout

Ingest notes live under `sources/{kind}/` (scaffold v0.1.1+). See [[Sources MOC]] and
`../../wiki_scaffold/docs/obsidian_setup.md` §8 for the full table.

| Folder | `source_kind` | Examples |
|--------|---------------|----------|
| `sources/discussions/` | `discussion` | Cursor conversations |
| `sources/papers/` | `paper` | Academic papers |
| `sources/articles/` | `article` | External articles |
| `sources/courses/` | `course` | Course notes |
| `sources/raw/` | — | L0 PDFs, clips |

Sibling wikis: see `../../README.md` under `Desktop/wikis/`.

Scaffold baseline: `../../wiki_scaffold/` — do not edit for daily finance notes.
