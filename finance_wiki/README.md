# finance_wiki

Finance knowledge wiki — macro cycles, IPO mechanics, China FX, cross-border investment access.

## Quick start

1. Install [Obsidian](https://obsidian.md)
2. *Open folder as vault* → **`Finance-Wiki/`** in this repo (not repo root)
3. Start at **[[Home MOC]]** → **[[Finance Topics MOC]]**
4. Enable **Dataview** plugin (manifests in `Finance-Wiki/.obsidian/`)
