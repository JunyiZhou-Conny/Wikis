# autoencoder_wiki

Autoencoder & representation-learning knowledge wiki.

## Quick start

1. Install [Obsidian](https://obsidian.md)
2. *Open folder as vault* → **`docs/`** in this repo
3. Start at **[[Home MOC]]**
4. Enable **Dataview** plugin

## Layout

```
autoencoder_wiki/
├── AGENTS.md            # project brain: hard rules, credibility + linking standard
├── README.md
└── docs/                # Obsidian vault root
    ├── maps/            # MOC hubs
    ├── concepts/        # atomic ideas connecting sources
    ├── sources/
    │   ├── discussions/
    │   ├── papers/      # require a credibility tier (A/B/C)
    │   ├── articles/
    │   ├── courses/
    │   └── raw/         # L0 PDFs + manifests
    └── 00-system/       # contract, domain profile, templates
```

**Skills + rule:** shared across all wikis at `../.cursor/` (`ingest-paper`, `distill-concepts`, `link-audit`, `daily-paper`).
**Sibling wikis:** `../finance_wiki/`, `../ml-hardware_wiki/` — see [wikis README](../README.md).
