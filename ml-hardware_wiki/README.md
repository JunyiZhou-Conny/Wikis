# ml-hardware_wiki

CPU/GPU hardware and model training knowledge wiki.

## Quick start

1. Install [Obsidian](https://obsidian.md)
2. *Open folder as vault* → **`docs/`** in this repo
3. Start at **[[Home MOC]]** → **[[Training Hardware MOC]]**
4. Enable **Dataview** plugin
